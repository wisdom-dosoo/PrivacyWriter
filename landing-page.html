How It Works: A Local-First AI Engine

1. Chrome's Built-in AI

We first use the native, hyper-fast AI APIs in Chrome for tasks like proofreading and rewrites. This is 100% local.

2. Gemini Nano Fallback

If a dedicated API isn't available, we use the on-device Gemini Nano model for more complex tasks. Still 100% local.

3. Optional Cloud (Pro)

Pro users can optionally connect their own Claude API key for maximum power. This is the only time data leaves your device.

Frequently Asked Questions

Is it really private?

Yes. By default, all AI processing happens on your device. We only send data to a third-party service (like Anthropic's Claude) if you are a Pro user and have explicitly enabled it by providing your own API key.

What is on-device AI?

It means the AI models run directly inside your Chrome browser. This is made possible by new technologies from Google (Project Gemini Nano). It's faster, works offline, and is completely private.
