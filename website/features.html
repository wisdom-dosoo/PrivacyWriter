Frequently Asked Questions

How does PrivacyWriter protect my privacy?

Unlike other writing assistants that send your text to cloud servers, PrivacyWriter processes everything locally on your device using Chrome's built-in AI. Your text never leaves your browser.

Is PrivacyWriter free to use?

Yes! The core features including grammar checking, rewriting, and summarization are completely free forever. We offer a Pro plan for advanced features like cloud sync and custom prompts.

Do I need an OpenAI or Claude account?

No. The extension runs standalone using local AI models. However, Pro users can optionally connect their own API keys if they prefer cloud-based processing for specific tasks.

Does it work on all websites?

PrivacyWriter works on almost any text input field in your browser, including Gmail, Google Docs, LinkedIn, Twitter/X, and more.

How do I install it?

Simply click the "Add to Browser" button, which takes you to the Chrome Web Store. Once installed, pin the extension icon to your toolbar for easy access.
